# roadmaps/ — 작업별(task) 로드맵

프로젝트 전체 로드맵은 상위 `roadmap.md`. 이 폴더는 **한 작업(task) 단위의 마일스톤 호**를 담는다 — `status.md` "다음 시작점"이 가리키는 task 브랜치(예: `tasks/spi-10ms`)와 1:1.

- 파일명: `<task>.md` (예: `spi-10ms.md`)
- frontmatter: `tags: [roadmap, task, <task명>, living-doc]`, `date`
- 구조: 프로젝트 `roadmap.md`와 동일 결(한 줄 요약 / 마일스톤 호 / 현재 위치→[[status]] 위임 / 남은 일정 / 환원 후보)을 task 범위로 축소.

컨벤션 전문은 root `wiki/CLAUDE.md` "로드맵 컨벤션" 절 참조. 실제 task 로드맵은 필요 시 생성한다.
