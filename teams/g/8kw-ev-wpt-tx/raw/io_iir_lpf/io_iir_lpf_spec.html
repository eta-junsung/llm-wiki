Digital Filter

1. 필터 구조 — Direct Form I, 2차 IIR

out0 = b0·in0 + b1·in1 + b2·in2 + a1·out1 + a2·out2
        ←── 피드포워드 (b계수) ───→  ←── 피드백 (a계수) ──→

차분 방정식 (코드 그대로):

out0 = b0·in0 + b1·in1 + b2·in2 + a1·out1 + a2·out2

Excel은 Ts = 1/40,000 (40kHz 기준 설계), 코드는 CLA 85kHz에서 동작 → a2 소수점 차이 발생

대칭 특성 (저역통과 증거):

  • b0 = b2 (대칭) → 선형 위상 저역통과 필터
  • b1 = 2 × b0 → Butterworth 2차 LP 특성

4. 코드 구현 매핑

// ① 입력 시프트 (in2 ← in1 ← in0 ← 새 ADC)
dsp_adc_LPFMON.LPF_Io_in2  = dsp_adc_LPFMON.LPF_Io_in1;
dsp_adc_LPFMON.LPF_Io_in1  = dsp_adc_LPFMON.LPF_Io_in0;
dsp_adc_LPFMON.LPF_Io_in0  = ADC_readResult(ADCARESULT_BASE, ADC_SOC_NUMBER1) - Vadc/2;
//                                                                                 ^^^^^^
//                           ADC 바이폴라 측정 중점 보정 (전류는 ±방향 모두 측정)

// ② 출력 시프트 (out2 ← out1 ← out0)
dsp_adc_LPFMON.LPF_Io_out2 = dsp_adc_LPFMON.LPF_Io_out1;
dsp_adc_LPFMON.LPF_Io_out1 = dsp_adc_LPFMON.LPF_Io_out0;

// ③ 필터 계산 (차분방정식 적용)
dsp_adc_LPFMON.LPF_Io_out0 =
    RX_adclpf_b0 * dsp_adc_LPFMON.LPF_Io_in0
  + RX_adclpf_b1 * dsp_adc_LPFMON.LPF_Io_in1
  + RX_adclpf_b2 * dsp_adc_LPFMON.LPF_Io_in2
  + RX_adclpf_a1 * dsp_adc_LPFMON.LPF_Io_out1
  + RX_adclpf_a2 * dsp_adc_LPFMON.LPF_Io_out2;

5. 필터 특성 요약

항목값
필터 종류2차 IIR 저역통과 (Butterworth 근사)
샘플링 주파수~85 kHz (CLA, FB 주파수 기준)
Excel 설계 기준40 kHz
차단 주파수 (추정)수십~수백 Hz 대역
목적Io ADC 스위칭 노이즈 제거
- Vadc/2ADC 바이폴라 측정 중점 보정

6. 코드와 Excel 계수 차이 원인

Excel a2 = −0.987472   (Ts = 1/40,000 = 25μs 기준 설계)
Code  a2 = −0.987512   (Ts = 1/85,000 ≈ 11.76μs 기준 재설계)
  • b 계수도 Excel은 반올림값 (1E-5), 코드는 정밀값 (9.75E-6) 사용
  • 실제 차단 주파수를 85kHz 동작 기준으로 미세 조정한 결과

7. 차단 주파수 정밀 확인 방법

Digital_filter.xlsx의 계수 셀(C3~C7)에 코드 정밀값 입력:

C3 = 0.00000975
C4 = 0.0000195
C5 = 0.00000975
C6 = 1.987473
C7 = -0.987512

→ sheet1 차트에서 주파수 응답 확인 가능
