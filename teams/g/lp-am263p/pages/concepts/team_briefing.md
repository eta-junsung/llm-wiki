## 1. 이 작업이 뭔가

**목표**: BP-CC3351을 LP-AM263P에서 구동 — WiFi scan/connect + BLE HCI 실동작.

**왜 이게 일인가**: TI는 이 WiFi 카드를 **LP-AM243(구형 보드)** 용으로만 공식 지원. LP-AM263P용 예제·드라이버 설정은 제공하지 않음.
→ AM243용 SDK 예제를 AM263P에 **직접 이식(포팅)**. 비공식 작업.

> **한 줄 비유**: 구형 PC에서 돌던 OS+WiFi 드라이버를 새 PC에 설치하는 것. 새 PC 전용 설치 가이드는 없음.

## 3. (과거) 가장 길게 막혔던 곳 — S3 (27라운드, 약 1개월+) ✅ 해결됨

> 이미 해결된 단계지만, "오래 막혔다 풀린 대표 사례"로 보고 시 참고.

**상황**: BIOS(SBL)는 USB(플래시)를 잘 읽고 OS(App)도 RAM에 떴는데, **뜬 OS가 자기를 띄운 그 USB를 다시 못 잡음.**
**원인**: BIOS와 OS 드라이버가 각자 USB 컨트롤러를 초기화하는데, OS 쪽 설정 파일(syscfg)의 **핀 번호가 틀려 있었음**. (구형 Quad-SPI ↔ 신형 Octal xSPI 8D DDR 방식 차이로 파악이 어려웠음)
**해결(R28)**: 핀 자동추천(`$suggestSolution`)으로 잘못 배치된 핀 7개를 **수동 고정(`$assign`)** 으로 교정 — 설정 한 줄 수정으로 즉시 인식.

## 4. 지금 어디 — S6 (이번주 R35~R38로 원인 좁힘)

### 무슨 상황이었나 (저번주)

"SPI 통신 무응답(`SPI not responsive`)" — PC(MCU) ↔ WiFi 카드 데이터선이 응답 없음. **원인이 우리 쪽(MCU)인지 카드 쪽인지 미상.**

### 이번주 진행 (계측으로 원인 위치 확정)

로직 분석기·계측으로 **원인 위치를 단계적으로 좁힘**:

1. **R35** — 클럭이 "0회"로 측정 → MCU가 전송을 안 하는 줄 알았음.
2. **R36** — 진단 마커로 확인하니 **전송 정상·클럭 ~16MHz 핀까지 도달**. R35 "0회"는 **계측 아티팩트**(아래 4-1)로 판명. 단 카드 데이터선(MISO)·인터럽트선은 응답 없음.
3. **R37** — **"MCU 마스터 정상 / 카드(NP) 무응답"** 판정 확정. 무게중심을 우리 쪽 → 카드 쪽으로 이동.
4. **R38** — 카드 측 **모든 출력핀(데이터·인터럽트 2계통·디버그 로그) 전부 침묵** 확인. 전원·리셋·레벨시프터·MCU SPI를 차례로 배제. → **"카드의 NP 코어가 아예 실행되지 않고 있다."**

### 4-1. 이번주 만난 문제 & 해결 (보고용)

- **문제**: R35에서 "SPI 클럭 0회"로 측정돼 잘못된 방향으로 갈 뻔함.
- **원인**: 로직 분석기 샘플링 속도(12.5MS/s)가 16MHz 신호를 잡기엔 너무 느림(샘플링 정리 위반).
- **해결**: 샘플링 속도를 **125MS/s로 올려 재측정** → 클럭 정상 확인. "계측 한계를 결함으로 오진"한 케이스.
- **부수 교훈**: 새 측정을 서두르기보다 기존 로그를 먼저 소진할 것(R36 데이터가 이미 판정에 충분했음).

### 4-2. 막힌 곳 (현재)

- **S6 막힘 — 원인은 "NP(WiFi 칩 코어) 미실행"으로 좁혀짐.** 전원·리셋·클럭공급·레벨시프터는 다 갖춰졌는데 칩의 4개 출력 계통이 전부 무활동 = 단일 핀 고장이 아니라 코어 자체가 안 도는 상태.
- **1순위 의심: 40MHz 크리스탈(Y1) 미발진** — 단, 아직 **추론(미측정)**. (크리스탈이 안 뛰면 코어가 클럭을 못 받아 실행 불가)

## 5. 다음 계획 (R39) — 어떻게 해결할 예정인가

- **오실로스코프(1GHz급 MSOX3104T)로 40MHz 크리스탈(Y1) 발진 여부 직접 측정.** 프로브 지점·주의(÷10 프로브 필수)는 확정 — **실측만 남음.**
- 판정: 40MHz 정현파 = 발진 OK(다른 원인 추적) / 양단 DC flat = **미발진 확정**(크리스탈·주변 회로 HW 조치).
- 병행: 칩 코어 생존(SWD/전류 측정)으로 크리스탈과 독립 교차검증.
